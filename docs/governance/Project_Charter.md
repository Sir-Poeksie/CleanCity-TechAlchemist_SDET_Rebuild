# CleanCity Waste Pickup Scheduler – Project Charter

**Purpose:** Define project objectives, scope, stakeholders, and governance framework.

**Project Name:** CleanCity Waste Pickup Scheduler  
**Prepared by:** Mpumelelo Theophilas Nxazonke  
**Date:** 2025-11-06  
**Version:** 1.0  

---

## 1. Project Vision

Deliver a scalable, secure, and accessible web and mobile platform for scheduling, tracking, and managing community waste pickups. 

---

## 2. Objectives

- Enable users to register, schedule, and track waste pickups.  
- Provide real-time notifications and dashboard analytics.  
- Ensure system compliance with GDPR, OWASP, and WCAG 2.1 standards.  
- Support community engagement and gamification.  

---

## 3. Scope

- **In Scope:** Authentication, Waste Pickup, Notifications, Admin Console, Community Features, Dashboard, API, Security, Accessibility, Performance.  
- **Out of Scope:** Third-party payment integration, IoT-enabled pickup hardware.
